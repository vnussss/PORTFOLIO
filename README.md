### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/vnussss/Portfolio.git
   cd Portfolio
   ```

2. **Open in browser**
   ```bash

   open index.html
   # Or use a local server

   ```

3. **View the portfolio**
   - Navigate to `http://localhost:8000` (if using server)

   - Adjust gradients and themes

   - Duplicate `.project-card` structure
   - Update images, titles, descriptions
   - Add your GitHub/demo links
